# CODSOFT — Machine Learning Internship

This repository contains my submissions for the CodSoft Machine Learning
Internship. Each folder is a self-contained task with its own notebook.

## Tasks completed

| # | Task | Approach | Folder |
|---|------|----------|--------|
| 1 | Movie Genre Classification | TF-IDF + Logistic Regression / Naive Bayes | [`Task1_MovieGenreClassification`](./Task1_MovieGenreClassification) |
| 2 | Credit Card Fraud Detection | Logistic Regression + Random Forest (class-balanced) | [`Task2_CreditCardFraudDetection`](./Task2_CreditCardFraudDetection) |
| 3 | Spam SMS Detection | TF-IDF + Naive Bayes / Logistic Regression | [`Task4_SpamSMSDetection`](./Task4_SpamSMSDetection) |

## How to run

1. Clone this repo:
   ```bash
   git clone https://github.com/<your-username>/CODSOFT.git
   cd CODSOFT
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Download the dataset for each task from the link given in the internship
   task PDF, and place the file in that task's folder (see the note at the
   top of each notebook for the expected filename).
4. Open the notebook (Jupyter or VS Code) and run all cells:
   ```bash
   jupyter notebook
   ```
