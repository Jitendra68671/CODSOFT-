# Task 1 — Movie Genre Classification

Predicts a movie's genre from its plot summary using TF-IDF text features
with Naive Bayes and Logistic Regression classifiers.

**Dataset:** download `train_data.txt` from the link in the task PDF and
place it in this folder (format: `ID ::: TITLE ::: GENRE ::: DESCRIPTION`).

**Run:** open `movie_genre_classification.ipynb` and run all cells.
