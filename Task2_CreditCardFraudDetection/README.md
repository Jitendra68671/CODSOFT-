# Task 2 — Credit Card Fraud Detection

Classifies credit card transactions as fraudulent or legitimate using
Logistic Regression and Random Forest, with class-imbalance handling.

**Dataset:** download the dataset from the link in the task PDF and save it
as `creditcard.csv` in this folder. The notebook auto-detects either the
classic anonymized schema or the richer transactions schema.

**Run:** open `credit_card_fraud_detection.ipynb` and run all cells.
