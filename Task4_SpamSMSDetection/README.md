# Task 3 (Task 4 in the PDF) — Spam SMS Detection

Classifies SMS messages as spam or ham (legitimate) using TF-IDF text
features with Naive Bayes and Logistic Regression.

**Dataset:** download `spam.csv` from the link in the task PDF and place it
in this folder.

**Run:** open `spam_sms_detection.ipynb` and run all cells.
